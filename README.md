# 🥘 Yurttaş Mutfakta

Yurt öğrencilerinin mutfak yönetimi, bütçe kontrolü ve yemek planlama süreçlerini optimize eden yapay zeka destekli akıllı asistan.

## 🚀 Proje Hakkında

"Yurttaş Mutfakta", kısıtlı bütçe ve sınırlı ekipmanla yaşayan üniversite öğrencilerinin mutfak deneyimini dijitalleştiren bir platformdur. Öğrencilerin elindeki kısıtlı malzemelerle yapay zeka kullanarak tarifler üretmesini sağlar, gıda israfını önler ve bütçe dostu alışveriş listeleri oluşturur.

### ✨ Temel Özellikler

*   **AI Destekli Tarif Üretici:** Elinizdeki malzemeleri girin, mutfak asistanı size uygun tarifi anında hazırlasın.
*   **Envanter & İsraf Yönetimi:** Buzdolabı ve kilerinizi takip ederek bozulmadan tüketim takibi yapın.
*   **Bütçe Dostu Alışveriş Listesi:** Öğrenci bütçesine uygun market listeleri ve fiyat optimizasyonu sağlayın.
*   **Oyunlaştırma & Sosyal:** Topluluk içi tarif paylaşımı, rozetler ve kullanıcı puanlaması ile mutfağı eğlenceli hale getirin.

## 🛠️ Teknolojiler ve Mimari

*   **Frontend Framework:** React / Vite (TanStack Start)
*   **Veritabanı & Kimlik Doğrulama:** Supabase
*   **Yapay Zeka Entegrasyonu:** LLM destekli akıllı tarif ve planlama motoru
*   **Stilleme:** Tailwind CSS
